Simulation times (ms) at default, 4x, 8x and 16x problem
sizes(Nx, Ny) compared between (parallelized) CPU and GPU.

        CPU         GPU         speedup
Def     500         564         0.89x
4x      8123        620         13.1x
8x      35381       941         37.6x
16x     147783      2093        70.6x

We can see strong speedup when using GPUs for the task compared to CPUs.
This speedup also becomes much more significant when we increase the problem size, as expected.
